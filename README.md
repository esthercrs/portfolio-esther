# Esther Cros – Portfolio

A personal portfolio website presenting my background, projects, and experience in **oceanography** and **bioinformatics**.

## Live site

**[esthercrs.github.io/portfolio-esther](https://esthercrs.github.io/portfolio-esther/)**

*(Update this URL if you use a custom domain or a different GitHub Pages path.)*

## Site structure

| Section | Contents |
|--------|----------|
| **Home** | Introduction, profile photo, and social links |
| **About** | Academic background, interests, and skills |
| **Projects** | Selected work and project highlights |
| **Publications** | Publications and related material |
| **Contact** | How to get in touch |

## Technologies

- **HTML** — Page structure and content  
- **CSS** — Layout, styling, and responsive design  
- **JavaScript** — Footer year, mobile navigation menu  

## Deployment

The site is hosted with **GitHub Pages** from this repository.

## Customization

Content and copy are maintained in the HTML files at the project root (for example `index.html`, `about.html`, `projects.html`). Edit those files directly, then commit and push; GitHub Pages will serve the updated site.
